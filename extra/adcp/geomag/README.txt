Magnetic variation/declination
------------------------------

#updated to use 2015 WMM data

Each model is valid for 5years

2015-2020 - 2015 file
2010-2015 - 2010 file

Older dates should incoporoate the IGRF model or use an online viewer for the calculations.

Calculates magnetic variation/declination for any latitude/longitude/altitude,
for any date. Uses the NOAA National Geophysical Data Center, epoch 2010 data.
